# eSmartLabs
This is a spring web application. Designed to automate daily activities of clinical pathology labs.

# Requirements
1. Java 1.7 and Above
2. Spring 4.3.1
3. Spring Security 3.2.3
4. Sitemesh 2.x
5. Bootstrap, JS, JQuery

# Features
1. Lab registration
2. Lab login
3. Tests management
4. Patients tracking
5. Reports generation
